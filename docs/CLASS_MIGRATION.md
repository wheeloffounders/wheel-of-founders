| `text-gray-400` | `text-muted` |

| `text-[#E2E8F0]` | `text-primary` |

| `bg-gray-100` | `bg-secondary` |
| `bg-gray-200` | `bg-secondary` |
| `border-gray-200` | `border-primary` |
| `border-gray-300` | `border-secondary` |

**Keep as-is (accent/brand):**
- `text-white` on coral/navy buttons
- `bg-[#ef725c]`, `bg-[#152b50]` for accent buttons
- `border-[#ef725c]`, `border-[#152b50]` for accent borders
