# Ideogram prompts — Founder Energy blog series

Use these for **hero cover images** (`coverImage` in each MDX file). Save outputs as JPEG in `content/blog/` using the filenames listed.

## Style guide (use on every prompt)

Append this block to each prompt for visual consistency with `/blog/brain-quits-at-7pm`:

```
Editorial magazine illustration, warm muted palette (cream, soft coral #ef725c accents, sage, charcoal), soft natural light, calm sophisticated mood, no text, no logos, no watermarks, 16:9 aspect ratio, high-end business lifestyle aesthetic, subtle grain, founder-parent energy when relevant
```

## Ideogram settings (suggested)

- **Aspect ratio:** 16:9 (hero)
- **Style:** Design / Illustration (or Realistic if you prefer photo-editorial—stay consistent across series)
- **Magic prompt:** On for first pass; tighten wording if it adds clutter
- **Negative prompt (if available):** `text, watermark, logo, cartoon, neon, stock photo smile, cluttered UI`
